\# Sales Data Analysis Dashboard



\## Project Overview



This project focuses on Exploratory Data Analysis (EDA), SQL analysis, and dashboard creation using a sales dataset. The objective is to extract meaningful business insights and present them through visualizations.



\## Dataset



The dataset contains the following fields:



\* Order\_ID

\* Order\_Date

\* Customer\_ID

\* Customer\_Name

\* Age

\* Gender

\* City

\* Product

\* Category

\* Quantity

\* Unit\_Price

\* Total\_Sales

\* Age\_Group

\* Month

\* Quarter

\## Project Workflow



1\. Cleaned and prepared the sales dataset.

2\. Performed Exploratory Data Analysis (EDA) using Python.

3\. Created visualizations to identify sales trends and business insights.

4\. Executed SQL queries to analyze the dataset.

5\. Built an interactive Excel dashboard with KPIs and charts.

6\. Documented the project and uploaded it to GitHub.



\## Key Insights



\* Identified top-performing product categories based on total sales.

\* Analyzed monthly sales trends to understand business performance.

\* Compared sales across different cities and genders.

\* Evaluated customer purchasing behavior by age group.

\* Developed an interactive dashboard for business reporting.



\## Conclusion



This project demonstrates practical skills in data cleaning, data analysis, SQL querying, and dashboard development. The insights generated can help businesses make informed decisions regarding sales performance, customer behavior, and product demand.
